== Description ==

Copy/pasting Amazon Widgets HTML tags is boring and causing trouble if you switch between HTML and rich text editor.
Now you can earn money with affiliate links without any pain and with XHTML standard compliant code.

This plugin is made for you if you use Amazon Affiliate links (aka Amazon Associates, Amazon Partenaires etc.):
* valid XHTML code
* minimalist shortcodes for easy copy/paste in your posts
* many exciting things to come, just check the Todo-List

== Installation ==

The plugin is very basic and made for my custom purpose. So for now, you have to manually configure the plugin once to fit your needs.

1. Rename the `config.php-dist` file as `config.php` and setup your data. Don't worry, everything is self-explained :
 1. `AWS_COUNTRY_TLD`
 1. `AWS_TRACKING_ID`
1. Simply upload the `amazon-widgets-shortcodes` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
